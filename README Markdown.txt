段落
空行を開ける

改行
行末に2つのスペースを入れる

強調
アスタリスク(*)またはアンダーバー(_)で囲う

強い強調
アスタリスク2つ(**)またはアンダーバー2つ(__)で囲う

コード表示
バッククォーテーション(`)で囲う

リスト
行頭にアスタリスク(*)か番号ピリオド(1.)を付ける

見出し
行頭の#の個数でH1からH6を表す
または
イコール行(=)で段落を区切るとH1、ハイフン行(-)で段落を区切るとH2

引用
行頭に大なり記号(>)を付ける

リンク
アングルブラケット(<>)でURLを囲う
または
[リンクのテキスト](リンクのアドレス "リンクのタイトル")

画像
先頭にぴっくりマーク(!)を付ける

水平線
3つ以上のハイフン(-)、アスタリスク(*)、アンダースコア(_)を並べる

エスケープ文字
\
`
*
_
{}
[]
()
#
+

.
